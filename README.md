# 요구사항 정리
## 기능 요구사항
- 자동차는 이름을 가지고 있다
  - 💡생성자로 지정할 수 있도록 구현
- 자동차는 움직일 수 있다.
  - 0~9 사이에서 rando 값을 구한 후, 4 이상이면 전진하고 3 이하면 멈춘다.
    - 💡전진 : 한 칸 이동한다
## 컨벤션
### 패키지 구문
- 열제한 120자
- 인덴드 depth는 2까지 허용
- 3항 연산자 비허용
- else 예약어 비허용
- 함수의 길이가 15라인을 넘어가지 않도록
  - 함수가 하나의 일만 잘 하도록 구현
